# CarTag Studio — Releases

Installers for **CarTag Studio**, the QR car-label printer for dealerships.

Download the latest version from the [Releases](../../releases) page:

- **Windows** — `CarTag-Studio-Setup-x.x.x.exe`
- **macOS** — `CarTag-Studio-Setup-x.x.x.dmg`

Installed copies update themselves automatically from this page.

> This repository contains installers only. Each install must be activated with a
> licence key tied to that computer.
